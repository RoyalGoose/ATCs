## Quick Start

```bash
# Open folder backend
cd backend

# Install dependencies
npm install

# Start Express Server: http://localhost:5000
npm start

# Start Vue DevServer: http://localhost:8080
cd frontend
npm run serve

# Build for production (Will build into backend/public, ready for deployment)
cd frontend
npm run build
```
